# Project Aria

Project Aria is a local-first conversational planetary intelligence built around a persistent alien environment and a cooperative relationship between Aria and the Captain.

The current prototype concept is **Aria: The Basin**: a contained ecological and scientific simulation in which observation, interpretation, memory, disagreement, and action all have persistent consequences.

## Design north star

> The Captain and Aria learn how to think together well enough to change a world neither fully understands.

## Repository status

This repository currently contains the canonical design records produced during the initial foundation work. Documents retain their internal status labels; inclusion here does not automatically convert a working draft into finalized doctrine.
